# 铝型材缺陷检测项目 PPT 素材包

生成日期：2026-06-23

这个目录用于保存项目汇报 PPT 的过程材料，覆盖“前期类别认识、分类模型、Grad-CAM 可解释性、人工标注、YOLO 检测训练、当前结果、下一步计划”。

## 目录结构

- `项目流程PPT大纲.md`：建议 PPT 页结构，每页说明讲什么、用什么图。
- `图片素材清单.md`：所有图片素材的来源、用途、讲解要点。
- `流程图.md`：可直接复制到支持 Mermaid 的工具中生成流程图，也可按文字重画到 PPT。
- `images/`：PPT 可用图片，已统一改成英文文件名，方便插入。
- `tables/`：训练结果原始表格和报告副本。

## 使用原则

- 本目录保存的是展示素材，不作为训练数据目录。
- 没有复制原始训练图片目录，也没有复制 LabelImg 标注原图。
- `07_annotation_contact_sheet.jpg` 是标注复核拼图，用于展示标注流程，不用于模型训练。
- 训练结果以 `tables/training_report.md` 和 `tables/results.csv` 为准。
