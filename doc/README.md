## 当前关键结论

当前建议采用循序渐进路线：

```text
先稳定整图分类
再做人工复核和样本清洗
再做缺陷区域粗定位
最后训练 YOLO 检测模型框出缺陷位置
```

现场第一版目标是辅助人工判断，不直接做全自动剔除。

推荐现场流程：

```text
固定相机拍照
模型判断是否疑似缺陷
屏幕显示结果图和缺陷提示
保存异常图片
工人决定是否拿走材料
```

后续如果需要精准框选缺陷区域，需要人工标注缺陷框，并训练检测模型。
