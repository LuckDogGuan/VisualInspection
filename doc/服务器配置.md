### 6.10 GPU 显存与 batch 调整建议

2026-06-23 训练时观察到 GPU 0 为 NVIDIA RTX A6000，显存约 `49GB`，训练中显存占用约 `4626MiB`，GPU 利用率约 `91%`。说明当前 `--batch 16` 对显存使用偏保守。

后续训练建议：

```text
优先尝试 batch 64
如果显存仍低于 20GB，可尝试 batch 96
如果出现 CUDA out of memory，退回 batch 32
如果 batch 64 指标明显变差，再回到 batch 16 做对照
```

推荐下一次训练命令：

```bash
python3 scripts/train_yolo_stage3.py \
  --data dataset_annotated_xxx/data.yaml \
  --model yolov8n.pt \
  --epochs 100 \
  --imgsz 640 \
  --batch 64 \
  --device 0 \
  --workers 8 \
  --project /data/home/guanjianxiong/visualinspection/yolo_stage3/outputs \
  --name server_train_xxx_b64 \
  --no-amp
```

注意：提高 batch 主要提升训练吞吐和稳定性，不保证 mAP50 一定提升。当前核心问题仍然是标注质量、类别定义和验证集分布。
