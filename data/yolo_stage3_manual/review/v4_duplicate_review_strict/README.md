## 规则

1. 同一组重复/近似重复样本中，优先保留 `batch_005`。
2. 如果没有 `batch_005`，优先保留批次号更大的样本。
3. 批次相同则优先保留人工修正或合并后的样本。
4. 只在同一类别内去重，避免不同缺陷类别被误合并。
5. 自动删除只接受同源文件名/来源路径强匹配。
6. 图像哈希只作为疑似重复提示，不自动删除。
